
# Modern React UI 
A tailwindcss powered modern react ui components library


## Installation

Install with npm

```bash
  npm install modern-react-ui
```
Install with yarn

```bash
  yarn add modern-react-ui
```
Install with pnpm

```bash
  pnpm add modern-react-ui
```
    
## Examples

```javascript
import { Clickable } from 'modern-react-ui'

function App() {
  return (
    <Clickable
      size="lg"
      text="Click Me"
      onClick={() => {
        console.log("hi");
      }}
   />)
}
```
